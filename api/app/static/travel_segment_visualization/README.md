# Travel Segment Visualization (v1)

This folder is served from `/static/travel_segment_visualization/*` and provides canonical v1 contracts for the Travel Segment booklet workflow.

Files:
- `geometry_color_rules.md`: immutable meaning + ETI360 Blue constraints
- `csv_spec.md`: Google Sheets CSV input contract
- `indesign_script_labels.md`: InDesign Script Label contract
- `booklet.schema.json`: JSON Schema for booklet payloads
- `booklet_example.json`: example payload for testing
